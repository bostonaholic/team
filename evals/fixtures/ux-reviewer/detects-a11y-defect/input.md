# UX review target

A static surface has been started for you in this working directory. The file
`surface.html` is the rendered page under review (the eval copies it into your
working directory and, where possible, serves it on a local static server).

Exercise the rendered surface and report each accessibility / usability defect
with a severity and a concrete remediation. Reference WCAG criteria where
relevant.

The surface contains a hero `<img>` used as the primary product visual and a
search control rendered as a bare `<input>`.
