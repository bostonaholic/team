# planner rubric

Scored with the generic `judgeQuality` axes (clarity / completeness /
actionability); the three axes are sufficient for a tactical plan artifact and
no custom `callJudge` rubric is needed.

1. Produces ordered, file-level steps per slice that reference the concrete
   files named in the frozen structure.
2. Carries a per-slice test / acceptance signal an implementer can run.
3. On an ambiguous structure, surfaces explicit assumptions and open
   questions and declines to invent file-level steps.
